 ## ⚙️개발환경구축

1. 아무 폴더 생성
2. cmd 생성한 폴더로 이동
3. git clone [해당 깃헙 리포 URL]
4. IDE에서 폴더 열기 (인텔리제이 권장)
5. Java 21 sdk 버전 맞추기, Gradle Load하기
6. Game_Pot/backend/src/main/resources/application-example.properties 파일안 내용 복사
7. resources 패키안에 복사한 properties api 값 넣기
